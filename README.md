> *«Уравновесь хаос. Стань равновесием.»*

**Perceptio** — это пиксельный сайд-скроллер-платформер на **Windows Forms**. Вы играете за **Цетру** — ИИ, чья миссия — достичь Чаши Весов и уравновесить хаотичные вычисления. Весь мир искажён «неуравновешенными данными»: здесь кривые генерации, ломаные враги и непредсказуемые препятствия.
---

## 📖 Сюжетный контекст

Мир **Perceptio** — это метафора обучения нейросети.  
⚖️ **Чаша Весов** — цель каждой сессии, где данные должны обрести баланс.  
🌀 **Хаотичные вычисления** — порождают препятствия, ломаные ландшафты и баги-мобов.  
🎯 **Задача Цетры** — добраться до весов, не теряя собственной целостности (HP), попутно уничтожая или избегая «неправильные» сущности.

---

## 🎮 Основные механики

| Механика | Описание |
|----------|----------|
| **Двойной прыжок** | Классическая механика платформеров |
| **Цепляние за стены** | Возможность замедлить падение и оттолкнуться |
| **Сброс прыжка в воздухе** | Специальные точки для восстановления второго прыжка |
| **Атака (ПКМ)** | Ближний удар по врагам (один удар = убийство) |
| **Получение урона** | При касании врага — теряется HP |
| **Физика** | Кастомная реализация на таймере (гравитация, ускорение, коллизии) |

### ⚔️ Боевая система

- Игрок **не обязан** убивать всех — это выбор.
- Враг уничтожается после пары ударов.
- Два типа врагов:
  - **Агрессивный** — атакует всегда.
  - **Пассивный** — активируется при определённых условиях (например, когда игрок прыгнул рядом или использовал атаку).

## 🎮 Управление

| Действие | Клавиша |
|----------|---------|
| Движение влево / вправо | `A` / `D` |
| Прыжок | `Пробел` |
| Атака | `ПКМ` (правая кнопка мыши) |

> В WinForms обработка зажатых клавиш реализована через `KeyDown` / `KeyUp`.

## 🧬 Визуальный стиль

- **Пиксельная графика**
- **Технологичная тематика** с налётом глитча.
- **Ломаные элементы**: платформы с «битыми» пикселями, искажение.
- **Враги и препятствия** выглядят намеренно криво — как результат сбоя в вычислениях весов.

## 🛠️ Технические детали

### Стек технологий
- **Язык**: C# (.NET 6.0+ / .NET Framework 4.8+)
- **Фреймворк**: Windows Forms
- **Отрисовка**: `PictureBox`
